# Notebooks Folder

This folder contains all Jupyter notebooks used for the project.

## Contents

- `raw_data_exploration.ipynb` → Initial exploration of raw data.
- `raw_data_report.ipynb` → Notebook used to generate raw data report.
- `data_cleaning.ipynb` → Notebook that implements data cleaning steps.
- `data_cleaning_report.ipynb` → Notebook used to generate cleaned data report.

### Notes

- Notebooks can be opened and executed using Jupyter Notebook.
- They are organized to follow the workflow: explore → clean → report.
