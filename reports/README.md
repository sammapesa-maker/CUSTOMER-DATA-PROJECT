# Reports Folder

This folder contains all generated reports for the project.

## Contents

- `raw_data_exploration.html / .pdf` → Report summarizing the raw data exploration.
- `raw_data_report.html / .pdf` → Detailed raw data analysis report.
- `data_cleaning_report.html / .pdf` → Report documenting the data cleaning process.

### Notes

- Reports are generated from the corresponding notebooks in the `notebooks/` folder.
- HTML files are interactive, PDF files are suitable for sharing or printing.
